> 为了能做到高精度快速计算，我们把所有的级数都变成整数运算，因此，如果我们希望计算到π的小数点后面n位，我就在> 上述公式两边乘以10的n次方，并全部运行整除运算（运算符号是“//”）。最后我们得到的一个π乘以10^n所形成的一个> 巨大的整数来表示计算结果（如果需要，可以通过字符变换把小数点加回去，这里就略过了）。
